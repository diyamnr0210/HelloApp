## UC2: Display Hello with Command-Line Argument
- Description: App accepts a name as an argument.
- Precondition: Run with a name argument.
- Output: Hello, [Name]!
## UC3: Display Hello with Command-Line Argument or Default Message
- Description: Handles missing arguments by defaulting to "World".
- Precondition: App runs with or without arguments.
- Output: "Hello, [Name]!" or "Hello, World!"
